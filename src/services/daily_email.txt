Налаштування процесу
Встановлення Python та необхідних бібліотек:

Переконайтеся, що у вас встановлений Python. Завантажити Python можна з офіційного сайту: https://www.python.org/downloads/
Встановіть бібліотеки smtplib та email (вони входять у стандартну бібліотеку Python і не потребують додаткової установки).
Налаштування Gmail акаунта:

Використовуючи Gmail як поштовий сервер, потрібно увімкнути опцію "Менш захищені програми" або створити "App Password" для підвищення безпеки. Інструкції можна знайти тут: https://support.google.com/accounts/answer/6010255
Запуск сценарію:

Збережіть вказаний вище сценарій у файл, наприклад daily_report_email.py.
Відредагуйте значення змінних sender_email, sender_password, recipient_email, subject, body та attachment_path.
Запустіть сценарій за допомогою командного рядка:
sh
Копіювати код
python daily_report_email.py
Автоматизація запуску:

Для автоматизації запуску сценарію щодня можна використати планувальник завдань, наприклад, cron у Linux або "Task Scheduler" у Windows.
Приклад налаштування cron (Linux)
Відкрийте crontab для редагування:

sh
Копіювати код
crontab -e
Додайте рядок для щоденного запуску сценарію о 9:00:

sh
Копіювати код
0 9 * * * /usr/bin/python3 /path/to/daily_report_email.py
Приклад налаштування Task Scheduler (Windows)
Відкрийте "Task Scheduler".
Створіть нове завдання і налаштуйте його на щоденний запуск о визначеній годині.
У полі "Дія" вкажіть шлях до сценарію daily_report_email.py.
Це допоможе вам автоматизувати процес надсилання щоденних звітів електронною поштою.
